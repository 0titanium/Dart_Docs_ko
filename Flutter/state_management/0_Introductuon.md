```
노트

Flutter를 사용하여 모바일 앱을 작성했는데 다시 시작하면 앱 상태가 손실되는 이유가 궁금하다면 Android의 상태 복원 또는 iOS의 상태 복원을 확인하세요.
```

Flutter를 탐색하다 보면 앱 전체에서 화면 간에 애플리케이션 상태를 공유해야 할 때가 있습니다. 취할 수 있는 접근 방식이 다양하고 생각해 볼 질문도 많습니다.
